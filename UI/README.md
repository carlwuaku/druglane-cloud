# Practitioners Portal

This is a web application built with the Angular framework.

## File Structure

The file structure of the project is as follows:

* `src/`
	+ `app/`
		- `features/` (features of the application)
			- `forms/` (forms feature)
				- `pages/` (pages of the forms feature)
					- `fill-form/` (fill form page)
						- `fill-form.component.html` (html template)
						- `fill-form.component.scss` (css styles)
						- `fill-form.component.ts` (typescript code)
					- `view-forms/` (view forms page)
						- `view-forms.component.html` (html template)
						- `view-forms.component.scss` (css styles)
						- `view-forms.component.ts` (typescript code)
			- `profile/` (profile feature)
				- `pages/` (pages of the profile feature)
					- `profile/` (profile page)
						- `profile.component.html` (html template)
						- `profile.component.scss` (css styles)
						- `profile.component.ts` (typescript code)
		- `libs/` (libraries)
			- `components/` (components)
				- `dashboard-tile/` (dashboard tile component)
					- `dashboard-tile.component.html` (html template)
					- `dashboard-tile.component.scss` (css styles)
					- `dashboard-tile.component.ts` (typescript code)
				- `file-uploader/` (file uploader component)
					- `file-uploader.component.html` (html template)
					- `file-uploader.component.scss` (css styles)
					- `file-uploader.component.ts` (typescript code)
				- `form-generator/` (form generator component)
					- `form-generator.component.html` (html template)
					- `form-generator.component.scss` (css styles)
					- `form-generator.component.ts` (typescript code)
				- `loading/` (loading component)
					- `loading.component.html` (html template)
					- `loading.component.scss` (css styles)
					- `loading.component.ts` (typescript code)
				- `select-object/` (select object component)
					- `select-object.component.html` (html template)
					- `select-object.component.scss` (css styles)
					- `select-object.component.ts` (typescript code)
		- `core/` (core module)
			- `services/` (services)
				- `auth/` (auth service)
					- `auth.service.ts` (typescript code)
				- `http/` (http service)
					- `http.service.ts` (typescript code)
				- `notify/` (notify service)
					- `notify.service.ts` (typescript code)
			- `models/` (models)
				- `user.model.ts` (typescript code)
		- `environments/` (environments)
			- `environment.development.ts` (typescript code)
			- `environment.production.ts` (typescript code)
		- `app/` (app module)
			- `app.component.html` (html template)
			- `app.component.scss` (css styles)
			- `app.component.ts` (typescript code)
		- `app-routing.module.ts` (typescript code)
		- `app.module.ts` (typescript code)
	+ `assets/` (assets)
	+ `environments/` (environments)
	+ `index.html` (html template)
	+ `main.ts` (typescript code)
	+ `polyfills.ts` (typescript code)
	+ `styles.scss` (css styles)
	+ `test.ts` (typescript code)
	+ `tsconfig.app.json` (typescript configuration)
	+ `tsconfig.spec.json` (typescript configuration)

IMPORTANT:To use ckeditor 5, unzip the ckeditor5.zip file into a ckeditor5 folder in the root of the application. make sure `"ckeditor5-custom-build": "file:ckeditor5",` is in package.json and run `npm i`.
